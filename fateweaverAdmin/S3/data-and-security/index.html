GDPR

The General Data Protection Regulation is something we all have to comply with. So here's where we stand. Any data you upload to StudentPath.co.uk is strictly confidential, that means we don't share it with anyone. Ever. StudentPath is a tool for your establishment to use and therefore you must get the correct permission for both adults and minors (In line with GDPR) before you upload any data to our service. You may already have this permission if you're storing any student or mentor information in a cloud based solution such as Google Drive.

If you recieve a request to remove data from someone on the system under GDPR you must notify us ASAP via contact@blooware.co.uk. We can then process the request accordingly as is our legal duty.
